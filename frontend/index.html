About Phishing Link Detection System

Phishing Link Detection System is a sixth semester project for “Detection of Phishing Website Using Machine (Deep) Learning”. It aims to help reduce phishing attack by helping internet users authenticate URL link by testing if it is phishing or legitimate. The progress of validating a Website URL for phishing or legitimate has gone through several layers of LSTM model. This project aims to develop a machine learning model that can detect phishing websites automatically. Phishing websites are fraudulent websites that mimic legitimate websites to trick users into providing sensitive information such as login credentials, credit card details, or personal information.


The machine learning model has be trained on a dataset of known phishing websites and legitimate websites to learn the characteristics and patterns that differentiate them. Features such as the website's URL, domain name, content will be extracted by the model and used as input to the machine learning algorithm.


Once trained, the model is be able to classify new websites as either legitimate or phishing based on their features. The model will be evaluated using metrics such as accuracy, precision to assess its performance.


The project involves data collection, preprocessing, model selection, and evaluation. The programming language used is Python, and popular machine learning libraries such as Tensorflow, numpy, pandas and matplotlib have be used for model development and training.


The goal of this project is to develop an effective and efficient solution for detecting phishing websites to improve online security and protect users from fraud and identity theft.
